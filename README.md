# 🩺 SkinDiseaseCheck

An offline Android application that leverages **Deep Learning** and **TensorFlow Lite** to classify common skin diseases from images selected from the device gallery.

The application performs **entirely on-device inference**, eliminating the need for an internet connection while providing fast, privacy-friendly predictions.

> **Disclaimer**
> This project is developed for educational and research purposes only. Predictions generated by the application are **not** a substitute for professional medical diagnosis or treatment.

# 📖 Overview

SkinDiseaseCheck demonstrates the deployment of a Deep Learning image classification model on Android using TensorFlow Lite.

The application allows users to select an image from their gallery, preprocesses the image locally, performs inference using a TensorFlow Lite model, and predicts the most probable skin disease class.

Since all processing occurs on-device, no user images are uploaded or stored.

# 🧠 Model Summary

The deployed model is a custom Convolutional Neural Network trained for multi-class skin disease classification.

Key techniques include:

- Convolutional Layers
- Max Pooling
- Batch Normalization
- Dropout
- Softmax Classification

The trained model was converted into **TensorFlow Lite (.tflite)** format for efficient mobile deployment.

For a complete explanation of the model architecture, training procedure, hyperparameter tuning, and technical decisions, see:

# 📱 Supported Diseases

The current model predicts the following skin conditions:

- Eczema
- Melanoma
- Atopic Dermatitis
- Basal Cell Carcinoma (BCC)
- Melanocytic Nevi
- Benign Keratosis-like Lesions (BKL)
- Psoriasis / Lichen Planus
- Seborrheic Keratoses
- Tinea / Ringworm / Fungal Infection
- Warts / Molluscum / Viral Infection
